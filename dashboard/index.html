Guide de configuration — 4 étapes

1. Google Sheets API
Crée un projet sur console.cloud.google.com → Active "Google Sheets API" + "Google Drive API" → Crée un compte de service → Télécharge le JSON → Partage ton Google Sheet avec l'email du compte de service

2. Gmail App Password
Google Account → Sécurité → Validation en 2 étapes → Mots de passe des applications → Génère un mot de passe pour "Domain Agent"

3. OpenPageRank (gratuit)
Inscription sur domcop.com/openpagerank → Récupère ta clé API gratuite (100 req/jour)

4. GitHub Secrets
Repo → Settings → Secrets and variables → Actions → New repository secret
Ajoute : GOOGLE_SHEET_NAME · GCP_SERVICE_ACCOUNT_JSON · GMAIL_FROM · GMAIL_TO · GMAIL_APP_PASSWORD · OPR_API_KEY
